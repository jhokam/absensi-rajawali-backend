## Run Locally

1. Get the code

```bash
  git clone https://github.com/jhokam/absensi-rajawali-backend
```

2. Install package

```bash
# bun
bun i

# npm
npm i
```

3. Run the code

```bash
# bun
bun dev

# npm
npm run dev
```
